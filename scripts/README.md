# Scripts

这里放本地开发脚本。

PowerShell：

- `init-db.ps1`：初始化本机 PostgreSQL 中的 `deckgo` 数据库
- `start-backend.ps1`：启动 Spring Boot 后端
- `start-web.ps1`：启动 Vue 工作台

Bash：

- `common.sh`：加载根目录 `.env` 和公共校验
- `init-db.sh`：初始化本机 PostgreSQL 中的 `deckgo` 数据库
- `start-backend.sh`：启动 Spring Boot 后端
- `start-web.sh`：启动 Vue 工作台
